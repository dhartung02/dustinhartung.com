# Reqon Case Study Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a new portfolio case study page at `/work/reqon` for a real, production-grade personal product — a self-hosted, AI-assisted job-search CRM spanning web, iOS/iPadOS, and Chrome — and link the homepage's existing "Reqon" Featured Work card to it.

**Architecture:** Mirrors `src/components/pulse/`, `src/components/catalog/`, and `src/components/opsystem/`: a `content.ts` single source of truth, a `sections/` directory, reuse of `usePrefersReducedMotion`/`revealAnimation` from `src/components/pulse/`. The centerpiece is a fully interactive, click-driven recreation of the real web dashboard (tabs, stat strip, action-needed grid, opportunity list, click-to-open detail panel, analytics tab) — unlike AI Product Operating System, click-driven interactivity is appropriate here because there's a real GUI to faithfully recreate. All data inside the mockup is synthetic; the one real asset used verbatim is the already-anonymous mobile-companion screenshot at `public/reqon/mobile-companion.png` (already committed to this worktree).

**Tech Stack:** Next.js 16 (App Router), React 19, TypeScript (`strict: true`), Tailwind CSS v4, Framer Motion, lucide-react, `next/image`. No test framework — verification per task is `npx tsc --noEmit` + `npm run lint`, with `npm run build` at integration points.

**Source of truth:** `docs/superpowers/specs/2026-07-07-reqon-design.md` — read it if anything in this plan is ambiguous.

**Critical privacy constraint, applies to every task below:** every company name, role title, salary figure, recruiter name, and status inside the mockup and the Chrome-extension illustration must be invented. Never use the real company name or real salary figure found in the real opportunity-detail screenshot during research, and never use the three real company names found in the real Chrome-extension screenshot during research.
